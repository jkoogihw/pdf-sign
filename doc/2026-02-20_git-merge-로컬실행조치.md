# Git Merge 충돌 해결 및 로컬 실행 환경 구축

## 작업 일시
- 날짜: 2026-02-20
- 이슈: Git merge 진행 중 상태 + 로컬 실행 불가

---

## 문제 상황

### 1. Git Merge 미완료
```
git status 결과:
On branch main
All conflicts fixed but you are still merging.
  (use "git commit" to conclude merge)
```
- 진행 중인 merge가 완료되지 않아 새로운 pull 불가능
- `.git/MERGE_HEAD` 파일이 존재하며 merge 상태 지속

### 2. 로컬 실행 불가
- `index.html` 및 `styles.css` 파일 누락
- 프로젝트는 정적 파일 앱이지만 필수 UI 파일이 없음
- HTTP 서버 실행 환경 부재 (Python, Node.js 미설치)

---

## 해결 조치

### Step 1: Git Merge 완료
```bash
git commit -m "Merge conflict resolution"
```
**결과**: merge 완료, working tree clean

### Step 2: 누락된 파일 복원
```bash
git checkout b2d3929 -- index.html styles.css
```
- 커밋 `b2d3929` (Merge pull request #8)에서 두 파일 복원
- 파일 인코딩 문제 해결 및 정상 로드 확인

### Step 3: HTTP 서버 구축
Python, Node.js가 없는 환경이므로 **PowerShell 기반 HTTP 서버** 구현

#### 생성 파일: `server.ps1`
```powershell
# 포트 8888에서 HTTP 서버 실행
# Content-Type 자동 설정 (HTML, CSS, JS, JSON)
# 404 처리 포함
```

### Step 4: 로컬 서버 실행
```bash
powershell -ExecutionPolicy Bypass -File server.ps1
```
**포트**: 8888  
**접속 URL**: `http://localhost:8888`
